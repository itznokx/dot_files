# DEPENDENCIES
- bash
- grep

# INSTALL
## Debian and Debian-like
- run `bash install.sh`
- after running `install.sh` script, run:
    > `nvim --headless -c 'autocmd User PackerComplete quitall'` -c 'PackerInstall' or `bash nvim_fs.sh`
